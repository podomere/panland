PS : Tip : Skills Test

At PanSift we recommend that organisations request an automated initial technical screen. By using a position description, CV, or candidate profile as a form of keyword currency we build custom simulations to test a practitioner's claimed skilling. You can also use predefined simulations or build your own simulations via drag and drop keywords. We call the process of testing candidates in real live scenarios "Simulation Based Screening". It allows for wider funnels and less bias whilst saving costs and reducing time-to-hire.
